## DESCRIPTION

Tangible Landscape is a collaborative modeling environment for
analysis of terrain changes which couples a scanner, projector
and a physical 3D model with GRASS.

The software consists of a plugin for GRASS and addon
[r.in.kinect](https://github.com/ncsu-geoforall/r.in.kinect)
for Kinect for Windows.

![Tangible Landscape plugin](tangible_landscape_dialog.png)

For all documentation and tutorials please refer to Tangible Landscape
[wiki](https://github.com/tangible-landscape/grass-tangible-landscape/wiki).
